Q: Do I need to change the frontend?
A: NO. All changes are backend only. Frontend stays the same.

Q: Will this break existing questions?
A: NO. New fields are optional. Existing data continues to work.

Q: How long does implementation take?
A: 30 minutes with copy-paste snippets. 1-2 hours if reading everything.

Q: Do I need to install new packages?
A: Only if using SMTP: npm install nodemailer
   Or if using SendGrid: npm install @sendgrid/mail
   For console logging (dev), no packages needed.

Q: What if email sending fails?
A: Failure is logged. Answer is still saved. emailError field tracked.

Q: Can students see each other's contact info?
A: NO. Public API excludes email/phone. Only mentors see contact details.

Q: How is phone validation secure?
A: Validates 10 digits only. Rejects sequential (1234567890), repeated (0000000000),
   and test numbers. Prevents garbage data in database.
